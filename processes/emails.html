Contactable Email addresses

This page attempts to be a directory of relevant email addresses relating to seL4. Public mailing lists allow anyone to sign-up to receive emails. Private mailing lists have a restricted set of receivers.

Public mailing lists:

  • announce@sel4.systems: For announcements about seL4. Sending to this list is restricted.
  • devel@sel4.systems: For asking questions about seL4.

Private mailing lists

  • moderation@sel4.systems: For requests or queries about community moderation. See seL4 Code of Conduct.
  • webmaster@sel4.systems: For website issues.
  • support@sel4.systems: For enquiries about professional (paid-for) support contracts.
