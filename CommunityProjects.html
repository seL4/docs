Application areas

The seL4 kernel is currently used in various projects from different application areas:

  • Automotive
  • Aviation, e.g.
    • SMACCM
    • CASE
  • Connected Consumer Devices
  • SCADA, e.g.
    • Laot
  • Spaceflight, e.g.
    • UNSW QB50 satellite
  • Virtualization
    • in general and especially for Linux

See also the list of Projects at Trustworthy Systems

OS Personalities and Frameworks

  • CamkES to build Trustworthy components
  • seL4 Core Platform personality for seL4
  • Genode
  • RefOS was created as reference OS
  • Neptune OS is a WinNT personality
  • UX/RT is a QNX-like and Linux-compatible OS personality for seL4

Languages and Runtimes

  • C is the default seL4 language
  • C++
  • Rust:
    • The Robigalia project: building a robust Rust ecosystem around seL4
    • ferros: A Rust-based userland which also adds compile-time assurances to seL4 development.
  • Python
  • WasmEdge for porting a Cloud-native WebAssembly Runtime to seL4
  • Ivory
  • Pancake
