The seL4 kernel is currently used in projects in the following application areas:

  • Automotive
  • Aviation (e.g., SMACCM)
  • Connected Consumer Devices
  • SCADA
  • Spaceflight (e.g., the UNSW QB50 satellite.)

The following platforms and language runtimes have been or are in the process of being ported to seL4:

  • CamkES
  • Genode
  • Linux (virtualised)
  • C
  • Ivory
  • Rust: The Robigalia project: building a robust Rust ecosystem around seL4
  • Python
  • C++
