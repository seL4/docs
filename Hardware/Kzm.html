KZM (Deprecated seL4 platform)

seL4 previously supported the KZM-ARM11-01 until version 12.1.0, which can also be simulated in qemu. Support for this platform has since been removed

The KZM is deprecated, ARMv11 Hardware which was used for the original seL4 verification. The latest verification platform is the SabreLite.
